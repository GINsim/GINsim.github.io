User documentation for GINsim

GINsim (see Naldi 2009) is a tool implementing the multilevel logical formalism. It leans on two main types of graphs: Logical Regulatory Graphs, which model regulatory networks, and State Transition Graphs, which represent their dynamical behaviour.
