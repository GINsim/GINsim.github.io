Strongly connected components

GINsim allows to compute the graph of the Strongly Connected Components (SCC graph) of an existing graph, through the Construct SCC graph option of the Tools menu.

Strongly Connected Components graph

Example of Strongly Connected Component Graph (bottom right) and the corresponding state transition graph (top left). The Selection Attribute tab in the bottom panel shows the content of the selected SCC (i.e. the list of nodes in the original graph).

Note that the Construct SCC Graph tool is often applied to State Transition Graphs but that it can also be applied to Regulatory Graphs (or more formally to any type of graphs).
