Styling graphs

The Style tab in the secondary window allows to change the visual properties of the graph elements.

GINsim uses styles to define visual properties of graph elements, in particular their color and shape. A style is a group of properties that can be applied to several elements. Nodes and edges have separate lists of styles.

The position of nodes, and the path followed by edges can be defined directly in the interactive view
