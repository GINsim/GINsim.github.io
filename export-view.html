Export graph view

The following exports consider only visual appareance of the graph, and can thus be applied to all types of graph in GINsim.

Unlike the Graphiz and BioLayout exports, the following formats preserve the graphical attributes defined within GINsim.

Image (PNG)

The graph view can be exported as an image in the PNG format.

SVG

The Scalable Vector Graphic (SVG) format is an XML-based format for representing vector graphics (www.w3.org/Graphics/SVG). Tools like Inkscape (inkscape.org), can be used to display, modify and export SVG files to virtually any image format.
