Lookup a path in the STG

The "search path" action enables the determination of paths between pairs of states of the state transition graph. Selecting the "search a path" action in the action menu opens a configuration dialog box, which allows the specification of a starting state, a target state , and optional intermediate states (using the "+" and "X" buttons to add/remove them). GINsim then performs a shortest path search. If several paths exist between these states, the first encountered shortest path is returned.
